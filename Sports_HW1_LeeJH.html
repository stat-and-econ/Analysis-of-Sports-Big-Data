  • This game took place on Tuesday (Day = Tue)
  • in front of 47,337 people (Attendance = 47337)
  • in Los Angeles (HmTm = LAN)
  • which lasted 2.7 hours (Duration = 162 minutes).

6. Constructing a similar figure for Aaron Judge

Using the same data (statcast2017.csv), simply replace Giancarlo Stanton with Aaron Judge.

x
judge_hr <- statcast2017 %>% 
  filter(player_name == "Judge, Aaron", events == "home_run") %>%
  mutate(is_fb = pitch_type %in% c("FF", "FT", "SI", "FC"))
plate_width <- 17 + 2 * (9/pi)
k_zone_plot <- ggplot(NULL, aes(x = plate_x, y = plate_z)) + 
  geom_rect(xmin = -(plate_width/2)/12, 
            xmax = (plate_width/2)/12, 
            ymin = 1.5, 
            ymax = 3.6, color=crcblue, fill = "lightgray", 
            linetype = 2, alpha = 0.01) + 
  coord_equal() + 
  scale_x_continuous("Horizontal location (ft.)", 
                     limits = c(-1.5, 1.5)) + 
  scale_y_continuous("Vertical location (ft.)", 
                     limits = c(1, 4))
k_zone_plot %+% judge_hr +
  aes(color = is_fb) + 
  geom_point() + 
  scale_color_manual("Type", values = c(crcblue,"gray60"),
                     labels = c("off-speed", "fastball")
  )

 

Comments on Judge's plot

  • Most of the balls he hit was in the strike zone. (Just like any other batters)
  • The off-speed balls and fastballs are roughly the same. (Just like Stanton)
  • More upper-side of the strike zone than lower-side of the zone.
